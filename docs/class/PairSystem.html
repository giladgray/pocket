Class: PairSystem

Defined in: src/utils/pair-system.coffee
Inherits: System

Overview

A System that accepts two requiredComponents arrays and passes both sets of resolved keys and components to its action function. Also support PairSystem.forEach to create a system that calls its action function for each pair of matching keys. This can be used to detect collisions or other interactions between keys.

Examples:

rocket.system PairSystem.forEach 'a-b', ['letter'], ['number'],
  (rocket, [keyA, letter], [keyB, number]) ->
    # function is called for each pair of [letter key, number key]
