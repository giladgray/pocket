Class: Rectangle

Defined in: src/utils/rectangle.coffee

Overview

A static Rectangle operations library.

A rectangle is simply an object with keys {x, y, width, height}. Rectangle.new(x, y, width, height) is shorthand for creating this object, but you can easily do it yourself too. Rectangle.new(x, y, size) will create a square where width == height.

All functions on this class are static and the constructor should never be used.

Examples:

r1 = Rectangle.new(1, 2, 3, 4)
r2 = Rectangle.centered(1, 1, 4)
r3 = {x: -1, y: -1, width: 4, height: 4}

Rectangle.equal(r2, r3)   # -> true
Rectangle.overlap(r1, r2) # -> true
